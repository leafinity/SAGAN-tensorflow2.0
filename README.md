# Self-Attention GAN
Tensorflow-2.0 implementation of ["Self-Attention Generative Adversarial Networks."](https://arxiv.org/abs/1805.08318).
## Usage

* train

    python main.py --train

* generate
    
    python main.py

the results will show in `./results`.
